## Requisitos

- Node.js (version 22)
- npm

## Project setup

```bash
$ npm install

$ npx prisma db push

$ npm run build
```

## Compile and run the project

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# e2e tests
$ npm run test:e2e
```

# Endpoint URL: http://localhost:3000/movies/awards/intervals

Para alterar o csv que é importado no banco de dados é só alterar o arquivo dentro da pasta src/assets.
